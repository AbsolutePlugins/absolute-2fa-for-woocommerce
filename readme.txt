A [Two Factor Authentication](https://wordpress.org/plugins/two-factor-authentication/) addon that will add 2fa settings page under WooCommerce's My Account Page.

== Description ==

Absolute 2fa For WooCommerce plugin seamlessly integrates into your WooCommerce login page implementing a two-factor authentication process! This will help you to prevent brute force attack for your WooCommerce shop. The plugin will provide you a straightforward usages experience as there in no configuration required. 
 
== Installation ==

If not installed, install [Two Factor Authentication](https://wordpress.org/plugins/two-factor-authentication/#installation) first.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
